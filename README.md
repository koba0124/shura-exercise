# shura-exercise

## 概要

解神という、謎解きを競い合うイベントがあります。

https://tokigami.jimdofree.com/

その中でも修羅という早押し勝負の練習のためのツールを作りました。

## 主な機能

* ローカルのPDFファイルを表示（アップロードはされません）
* 30秒のカウントダウン
* 解答ボタンを押すと、問題が消える
* 解答入力時の残り時間を記録

## サイト

https://koba0124.github.io/shura-exercise/
